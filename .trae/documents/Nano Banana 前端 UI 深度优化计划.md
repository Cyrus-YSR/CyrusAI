# 前端 UI 深度优化计划 - Nano Banana 2.0

针对您提出的需求，我制定了以下 5 步优化方案，旨在解决样式缺陷并提升整体视觉质感。

## 1. 修复输入框自动填充变白问题
**目标**: 解决浏览器记住密码后，输入框背景强制变白的问题。
**文件**: `vue-frontend/src/App.vue`
**方案**:
- 在全局样式中添加 `input:-webkit-autofill` 覆盖规则。
- 使用 `box-shadow: 0 0 0 1000px #2a2a2a inset` 强制覆盖浏览器默认背景。
- 设置 `-webkit-text-fill-color: #fff` 保持文字可见。

## 2. 注册页面 UI 翻新
**目标**: 将遗漏的注册页更新为最新的黑黄主题。
**文件**: `vue-frontend/src/views/Register.vue`
**方案**:
- 背景改为纯黑 + 新版动态特效。
- 卡片改为深色磨砂玻璃质感，带黄色微光边框。
- 按钮和输入框样式与登录页保持一致（黄色主色调，大写粗体）。

## 3. 全局动态背景特效
**目标**: 替换单纯的黑色背景，增加科技感和活跃度。
**文件**: `Login.vue`, `Register.vue`, `Menu.vue`, `AIChat.vue` (或统一在 App.vue)
**方案**:
- 实现 **"赛博网格 (Cyber Grid)"** 动画：深灰色的网格线在黑色背景上缓慢移动。
- 叠加 **"呼吸光晕"**：中心或随机位置出现微弱的黄色脉冲光斑。
- 视觉效果类似 "暗网流动" 或 "黑客帝国矩阵（黄色版）"，保持低调不抢眼。

## 4. 聊天界面圆润化与玻璃质感
**目标**: 消除过于方正的布局，增加通透感。
**文件**: `vue-frontend/src/views/AIChat.vue`
**方案**:
- **悬浮布局**: 给左侧列表和右侧聊天区增加外边距 (margin) 和圆角 (border-radius: 24px)，让它们看起来像悬浮在背景上的玻璃面板。
- **玻璃拟态**: 使用 `backdrop-filter: blur(20px)` 和半透明黑背景，让底部的动态网格隐约可见。
- **输入框优化**: 增加圆角，改为全胶囊形状或大圆角矩形。

## 5. 历史会话列表 "充能" 特效
**目标**: 鼠标悬浮时产生能量充盈的视觉反馈。
**文件**: `vue-frontend/src/views/AIChat.vue`
**方案**:
- 增加 CSS 动画：当鼠标悬浮在 `.session-item` 上时：
  - 边框瞬间变为亮黄色。
  - 产生向内和向外的黄色发光阴影 (`box-shadow`)，模拟能量过载。
  - 文字轻微右移，增加交互感。

确认计划后，我将按顺序执行这些修改并重新构建前端容器。
